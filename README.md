# Information Technology Textbook

English study textbook for **Cand.it. Digitalisering og applikationsudvikling** (Aalborg University), semesters 1–4: theory, principles, references, code examples, and lesson audio per module.

**Live site:** [https://sbf-developer.github.io/information-technology-textbook/](https://sbf-developer.github.io/information-technology-textbook/)

**UI language:** English · **Course titles:** Danish (official module names)

## Features

- Layered reading paths: plain-language intro, zero-to-mastery, lessons, core subject guides, tutorials
- Syntax-highlighted code blocks
- Web Speech API lesson audio (browser-dependent voices)
- In-browser PDF export of the full book

## Run locally

```powershell
npx serve . -l 5196
```

Open http://localhost:5196
